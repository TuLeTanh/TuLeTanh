## Featured Projects

### ⚽ [YOLO Football Analysis Pipeline](https://github.com/TuLeTanh)
> Computer vision pipeline for player/ball detection and tracking in football match footage

- Player & ball detection using **YOLOv8 / YOLOv5x**, tracked frame-to-frame with **ByteTrack**
- **K-means clustering** on jersey colors for automatic team classification
- **Optical flow** for camera motion compensation, stabilizing tracking under camera pans
- Designed to feed real positional data into downstream analytics (heatmaps, formations)

`Python` `YOLOv8` `ByteTrack` `OpenCV` `K-means` `NumPy`

---

### 😀 [Facial Emotion Recognition](https://github.com/TuLeTanh/Emotion-Recognition)
> Deep learning model for facial emotion classification (Kaggle dataset)

- Analyzed accuracy/generalization limitations of a baseline **VGG-style CNN**
- Designed a **ResNet-mini** architecture (residual blocks) using the **Keras Functional API** to address vanishing-gradient and overfitting issues
- Iterated on architecture based on empirical error analysis rather than default templates

`Python` `TensorFlow` `Keras` `CNN` `ResNet`

---

### 🏆 [Football Hub](https://github.com/TuLeTanh)
> Full-stack football score tracking app with live scores, standings, and knockout brackets

- **FastAPI + SQLAlchemy** backend with a provider/normalizer pattern pulling from the football-data.org v4 API
- **APScheduler** cron jobs for scheduled data refresh; live-score gating and animated score updates
- **Next.js + TypeScript** frontend: Overview, Standings, and Match Detail (Timeline, Stats, Lineups)
- Mock data layer enriched with real World Cup 2026 squad data (48 teams × 26 players), with formation-aware starter selection

`FastAPI` `SQLAlchemy` `Next.js` `TypeScript` `PostgreSQL` `APScheduler`
